# official-soccerodds-archive

这是一个用于归档和发布多个独立 HTML 页面的仓库。页面内容涵盖足球相关数据展示、信息整理与辅助工具，所有页面均为静态 HTML 格式，可直接在浏览器中打开查看。

## 项目简介

本仓库的主要目的为长期保存和公开提供一系列自制的 HTML 页面。这些页面不依赖特定后端服务，也不针对任何单一网站或域名，可作为独立资料或工具页面供有需要的用户参考。

## 目录说明

仓库根目录下按主题或用途分设若干文件夹，每个文件夹内包含对应的 HTML 文件及相关资源（如 CSS、JavaScript 或图片等）。具体目录结构如下：

```
official-soccerodds-archive/
├── folder-1/          # 第一类页面存放目录
│   ├── page1.html
│   └── page2.html
├── folder-2/          # 第二类页面存放目录
│   ├── subfolder/
│   └── index.html
└── README.md          # 本文件
```

各目录的详细说明请参见对应文件夹内的 `README.md`（如有）。

## 页面归档说明

- 所有页面均为静态 HTML，无需服务器环境即可运行。
- 页面内容可能包含数据表格、可视化图表或交互式元素，但均不收集任何用户信息。
- 部分页面可能包含外部链接，仅用于引用公开资料或标准库（如 Chart.js、D3.js 等）。
- 本仓库不保证页面中数据的实时性与准确性，也不对页面内容的使用后果负责。
